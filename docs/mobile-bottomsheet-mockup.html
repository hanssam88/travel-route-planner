바텀시트 인터랙션 설계

상태높이동작지도
Peek (접힘) ~140px 헤더(제목+요약칩)만 노출. 위로 스와이프하면 Half로 전환 전체 화면, 모든 마커+폴리라인 보임
Half (절반) ~55% 리스트 스크롤 가능. 위로 스와이프→Full, 아래→Peek 상단 45%에 축소 표시, 선택 마커 자동 포커스
Full (전체) ~85% 전체 리스트 탐색. 아래로 스와이프→Half 상단 15%에 미니맵 형태
카드-지도 연동 (기존 유지) - 카드 탭 → 지도 해당 마커로 카메라 이동 + 마커 하이라이트
- 마커 탭 → 바텀시트가 Half로 열리며 해당 카드로 자동 스크롤
- 바텀시트 상태와 무관하게 양방향 동기화 작동
